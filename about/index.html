基于NB-IoT的物联网可视化平台

时间：2020-08 ~ 2020-09

负责：前端开发

技术栈：Vue + Vue Router + Vuex + Echart + JWT

Github:https://github.com/LeBronChao/Big-Data-Visualization

  • 本人承担工作：单人完成Web前端开发和小程序开发，前后端接口路由、数据结构定义，联调及Web服务部署。

  • 软件功能：

  1. 基于JWT鉴权的登录验证，注册。
  2. 基于echarts的数据可视化仪表、折线图等基于百度地图的位置数据可视化、轨迹数据可视化等。（显示传感数据、位置数据）
  3. 软件开关控制实体设备。
  4. 根据用户选择，查询传感数据并动态生成折线图。
  5. 应用场景模块，开发了校园安全预警模块，实时显示今日、上周、上月预警数，可通过日期查询该日期的所有预警信息，选择预警可显示预警视频、预警位置等，此外还可生成年、月、周的预警数折线图观察预警数走向及趋势。
  6. 用户管理模块，修改用户信息、密码等，以及管理员的用户管理（修改所有用户的信息及密码）。
  7. 设备管理模块（开发中）。
  8. 小程序端复现设备控制模块、实时传感数据模块等。

校公文通系统

时间：2020-12 ~ 2020-12

负责：全栈开发

技术栈：React + React Router + Node + Express + Mysql + Sequelize ORM + JWT + MD5

Github：https://github.com/LeBronChao/Management-of-documents

访问地址：http://doc.lebronchao.com/

  • 本人承担工作：Web前端开发、后端开发、前后端接口设计、数据库设计、测试和服务部署。

  1. 路由守卫：自封装路由守卫组件，非本站用户只可访问登录、注册页。
  2. 公文浏览：本站用户可浏览所有公文。

  8. 用户管理：实现管理员对用户的全新和信息的增删改查。管理员权限设层级，实现区分管理员权限范围。
  9. 数据加密：通过MD5算法，实现对用户敏感信息的加密。
